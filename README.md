# SmartChats

A modern, real-time chat application built with **React, Node.js, Express, and MongoDB**.

SmartChats allows users to communicate instantly through a clean and responsive interface. It is designed for students, developers, and anyone who wants a simple and efficient real-time messaging platform. The application supports authentication, live messaging, and message history storage.
